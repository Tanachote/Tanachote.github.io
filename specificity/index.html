WHAT IS CSS-SPECIFICITY?

CSS-Specificity คือสิ่งที่บ่งบอกถึงลำดับความสำคัญของ selecter เพื่อกำหนด style ให้กับแต่ละ element ที่จะให้แสดงผลบนหน้าเว็บ

การคำนวณค่า Specificity

ในการคำนวณค่า Specificity นั้นจะขออนุญาตแบ่งประเภทการเขียน style ออกมาดังนี้โดยให้มองว่าในกลุ่มเดียวกันให้มีค่า specificity เท่ากัน

1) 1000 --มีค่า--> Inline style
2) 100 ---มีค่า--> ID selectors
3) 10 ----มีค่า--> Class selectors, Attributes selectors, Pseudo-classes
4) 1 -----มีค่า--> Type selectors, Pseudo-elements
5) 0 -----มีค่า--> Universal selector, Combinators, Negation pseudo-clas

ซึ่งถ้าเขียนแบบรูปด้านล่างจะได้ค่าดังนี้

ด้วยเหตุนี้จึงทำให้บางครั้งแม้ว่าจะมีการอ่านโค้ดจากบนลงล่างแต่ก็จะเลือกทำตัวที่มีความสำคัญมากที่สุด

ตัวอย่าง


ไฟล์ index.html ที่มีการเชื่อมโยงกับ style2.css และ style1.css
โดยจะเห็นว่าต่อให้มีการเขียน "ul li a" ทั้งใน 2 style แต่ไม่ถูกเลือก
มาแสดงเนื่องจาก "ul li .active" มีความสำคัญกว่า


ไฟล์ home.html ที่มีการเชื่อมโยงกับ style1.css และ style2.css
โดยจะเห็นว่ามีการเรียกแบบตามลำดับโดยจะเลือก style1.css มาเนื่องจากค่าความสำคัญของตัวกำหนด style นั้นมีค่าเท่ากันและ style1.css คืออันที่ถูกเรียกหลัง style2.css
